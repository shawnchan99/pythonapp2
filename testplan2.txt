================================================================
Use Postman (GUI Tool):

1. 	Download Postman.
2. 	Create a new request.
3. 	Set method to POST
4. 	Enter URL: http://localhost:8082/submit 
5. 	In the Body tab, choose: x-www-form-urlencoded
6. 	Add a key "name" with value "Alice".
7. 	Click Send — you’ll see the response.

response: {"message":"Thanks, Alice"}

================================================================
Use BASH curl:

Basic Authentication test
============================
curl -u admin:secret http://localhost:8082/secure-data


Query Parameters test
============================
curl "http://localhost:8082/greet?name=Alice"

response: {"message":"Hello, Alice!"}


POST test
============================
curl -X POST http://localhost:8082/submit \
     -d "name=Alice"

response: Thanks, Alice


JSON test
============================
curl -X POST http://localhost:8082/submit \
     -H "Content-Type: application/json" \
     -d '{"name": "Alice"}'

response: {"message":"Thanks, Alice"}
